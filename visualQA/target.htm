Introduction

This program can be used to visually compare and simultaneously scroll through two HTML files. To start either open a Source and Target file (File > Open), to compare multiple files in two different folders create a project (File > New project...).

The program works best when you use the hot-keys to scroll through the pages.

  • Use Up and Down and PageUp and PageDown to synchronize scrolling
    (Home and End can be used to jump to the top and bottom of the documents)
  • Use Ctrl-Up and Ctrl-Down and Ctrl-PageUp and Ctrl-PageDown to scroll the Source (left side)
  • Use Alt-Up and Alt-Down and Alt-PageUp and Alt-PageDown to scroll the Target (right side)
