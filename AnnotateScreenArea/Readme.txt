HOW TO USE
Select the area you want to annotate by moving the dialog (resize the dialog if required)
Click Set (you can now move the dialog window to another location)
Add your annotations (right-click to change tool)
Click Save to store the (annotated) area
Click Update to refresh the original area with new contents

NOTES
When entering text you use the mouse to position the text box.
To *set* the text press Enter. To insert a line-break use Shift-Enter.

CONFIGURATION
Right-click lower part of dialog to set output folder.
